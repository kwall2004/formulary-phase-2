Requesting an ID Card

If your Health Plan of Michigan identification card is lost or damaged, you can request a new one on the Request ID Card page.

  1. Click the Request ID Card menu bar on the left side of the screen.
    The Request ID Card page opens with your Medicaid id number and name displayed    .

  2. At the bottom of the page, the question Are you sure you want to request and ID Card appears. If you do, click Yes.
    A message appears telling you that your request was submitted successfully and that your new id card should arrive in approximately two weeks.
